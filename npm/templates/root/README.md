# Museon CLI

This dependency-free launcher selects the exact-version Museon native package
for the current operating system. Install it with:

```bash
npm install --global @museon/cli@{{VERSION}}
```

No npm lifecycle script or runtime download is used. See the
[installation guide](https://github.com/Museon-AI/museon-cli/blob/v{{VERSION}}/docs/install.md)
for supported platforms and the immutable GitHub/uv fallback.
